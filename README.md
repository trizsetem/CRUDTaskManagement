# crud_task_management

## DESCRIÇÃO
Aplicação desenvolvida em Flutter com uso dos plugins sqflite e path_provider, implementando um sistema de cadastro de tarefas profissionais.
O app permite criar, visualizar, editar e excluir tarefas, salvando todos os dados em um banco SQLite local.
